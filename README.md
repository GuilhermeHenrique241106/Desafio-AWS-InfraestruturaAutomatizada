Desafio de Projeto: Automatizando Infraestrutura com CloudFormation

O que eu aprendi neste laboratório:
- O que é Infraestrutura como Código (IaC) e como ela evita erros manuais.
- Como o AWS CloudFormation utiliza arquivos de texto (JSON/YAML) para criar recursos de forma automática.
- O conceito de Stacks (Pilhas), que agrupam o servidor e suas configurações de segurança em um só lugar.

Passo a passo executado:
1. Acessei o console do CloudFormation na AWS.
2. Utilizei um modelo de exemplo fornecido pela própria plataforma (LAMP Stack).
3. Configurei os parâmetros obrigatórios e acompanhei a criação dos recursos.
4. A Stack foi criada com sucesso!
